# StayAlive

## Objetivo
Dashboard com análise de dados via CSV.

## Stack
- Next.js 14 + React 18 + TypeScript
- Supabase (project ID: xgbbgrttqtmrnytaspxk)
- Recharts + papaparse + Vitest
- Vercel (deploy via GitHub)

## Comandos
- Dev: `npm run dev`
- Build: `npm run build`
- Testes: `npx vitest`

## Regras
- Componentes funcionais com hooks, sem classes
- Tailwind only, sem CSS custom
